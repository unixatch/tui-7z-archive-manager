## COMMAND LINE PARAMETERS

### --path, /path, -p, /p:

Sets the archive file path

### --pagesize, /pagesize, -ps, /ps

Sets the amount of things that prompts can show at once

### --yes, /yes, -y, /y:

Auto-confirm deletions

### --help, /help, -h, /h, /?:

Shows the help message

### --version, /version, -v, /v:

Shows the installed version
